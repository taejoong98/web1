HTML


strong = 글씨 진하게 표시(강조)
u = under line(밑줄)
h = heading(제목) h1~h6 까지
br = line breaker(줄바꿈)
p = paragraph(단락)

p로 위에 공백을 더 넓힐수 있다.
li = list(목차), ol = ordered list(앞에 번호부여), ul = unordered list(동그라미)
img = image(이미지를 나타내는 태그) 속성 : src = source(불러오기) width = 사진크기
<,img src="파일명 또는 웹페이지 주소" width="100%">
a = anchor(닻) 정보의 바다에 정박한다는 의미의 시적인 표현, 링크
링크 거는 방법 <,a href="주소"> 새 툴로 열고 싶으면 "주소"다음 <,target="_blank"> 붙이기
커서 올렸을때 툴팁으로 제목 보일려면 title="제목"까지 붙이기 href = HyperText Reference
